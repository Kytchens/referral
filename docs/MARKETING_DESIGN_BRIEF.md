# Marketing Design Brief: Kytchens Recruitment Campaign

**Prepared by:** Product Team
**Date:** 26 March 2026
**Version:** 1.0

---

## 1. Objective

Drive blue-collar kitchen staff to the Kytchens recruitment web app through offline and digital channels. Every touchpoint should make one thing clear: **"Achha kaam, achha paisa, abhi apply karo."**

---

## 2. Target Audience

- **Who:** Kitchen helpers, cooks, delivery riders, warehouse/retail workers looking for stable employment in Mumbai
- **Age:** 18–40
- **Income:** Currently earning ₹8,000–₹18,000/month, looking for ₹14,000–₹25,000
- **Digital behavior:** WhatsApp 3-4 hours/day, YouTube, Instagram Reels. Comfortable scanning QR (trained by Paytm/PhonePe). Limited English.
- **Mindset:** "Kitna milega? Kahan hai? Pakka kaam hai?"

---

## 3. Key Message Hierarchy

Every piece of marketing collateral must follow this hierarchy:

| Priority | Message | Example |
|----------|---------|---------|
| **1st** | Salary / Earnings | "₹14,000 – ₹25,000/month" |
| **2nd** | Role | "Kitchen me kaam karo" |
| **3rd** | No experience needed | "Fresher bhi welcome!" |
| **4th** | Action | "Abhi scan karo / Apply karo" |

**Salary is always the headline.** This is what blue-collar workers care about first. Everything else is secondary.

## 4. Brand Guidelines for Marketing

### Colors
| Color | Hex | Usage |
|-------|-----|-------|
| Kytchens Orange | `#FF5500` | Headlines, CTA buttons, accents |
| Cream | `#FDF6F0` | Backgrounds |
| White | `#FFFFFF` | QR code background, text cards |
| Dark | `#1A1A1A` | Body text |

### Typography
- **Headlines:** Poppins Bold — large, punchy, readable from 2 meters
- **Body:** DM Sans — clean, legible at small sizes
- **Minimum print size:** 24pt body, 48pt+ headlines on A3 posters

### Tone of Voice
- Hinglish always (Roman script Hindi)
- Short, direct sentences — max 8-10 words per line
- Conversational — like a friend telling you about a job, not a corporate announcement
- Use emojis sparingly in digital — 💰📍✅ for visual markers only

### Photography
- **Real kitchen photos** — actual Kytchens kitchens, real staff in uniform
- **Never use stock photos** — workers can tell immediately, kills trust
- Show people working, smiling, in action — not posed corporate shots
- Uniform visibility builds legitimacy — "Yeh asli company hai"

## 5. Deliverables

### 5.1 QR Code Poster (Print — Primary Channel)

**Purpose:** Physical posters at kitchen locations, bus stops, chai tapris, PG notice boards. This is the #1 candidate acquisition channel.

**Sizes:**
- A3 (primary) — for indoor walls, notice boards
- 2ft × 3ft flex banner — for outdoor/entrance placement

**Layout (top to bottom):**

```
┌──────────────────────────────────┐
│         KYTCHENS LOGO            │
│                                  │
│     🍳 WE'RE HIRING! 🔥         │
│                                  │
│  ┌────────────────────────────┐  │
│  │   ₹14,000 – ₹25,000/month │  │  ← LARGEST TEXT (salary)
│  └────────────────────────────┘  │
│                                  │
│  Kitchen me kaam karo!           │  ← Role
│  Fresher bhi welcome hai!        │  ← No experience needed
│                                  │
│  ✅ Full Training                │
│  ✅ PF + ESIC                    │  ← Benefits strip
│  ✅ Career Growth                │
│  ✅ Incentives + Bonus           │
│                                  │
│  ┌──────────┐                    │
│  │          │  ← Yahan scan     │
│  │  QR CODE │     karo!         │  ← QR with instruction
│  │          │  📱               │
│  └──────────┘                    │
│                                  │
│  Ya WhatsApp karo: 90821 83834   │  ← Fallback phone number
│  ya type karo: jobs.kytchens.com │  ← Fallback URL
│                                  │
│  Apne dost ko bhi batao! 👥     │  ← Referral nudge
└──────────────────────────────────┘
```

**QR Code Specifications:**

| Spec | Requirement |
|------|-------------|
| Minimum size | 4cm × 4cm (for low-end phone cameras) |
| Color | Dark on white background — maximum contrast |
| Error correction | Level H (30%) — survives dirt, wear |
| Content | Short URL → `jobs.kytchens.com` (keeps QR simple, fewer dots = easier scan) |
| Quiet zone | Minimum 4-module white border around QR |
| Fallback | WhatsApp number + typed URL below QR |

**QR Scanning Instructions (MUST include):**
- Visual cue: small phone icon pointing camera at QR
- Text: "📱 Yahan scan karo" with arrow pointing to QR
- Never assume workers know what QR is — always show + tell

**Testing Before Print:**
- Test scan on Redmi 9A, Realme C3, Samsung Galaxy M12 (common low-end phones)
- Test in low light conditions (kitchens are often dimly lit)
- Test at intended viewing distance (1-2 meters)
- Test with dirty/slightly crumpled paper

**Placement Guide:**

| Location | Why it works |
|----------|-------------|
| Next to time-punch machine | Workers see it daily, dwell time |
| Canteen / lunch area | Relaxed state, time to scan |
| Inside bathroom doors | Captive audience, high attention |
| Near water cooler | Brief pause, phone in hand often |
| Building entrance/exit | High traffic, visible to passersby |
| Chai tapris nearby | Workers hang out, scroll WhatsApp |
| Bus stops in target areas | Waiting time = scanning time |
| PG/hostel notice boards | Where workers live, shared amongst roommates |

### 5.2 WhatsApp Broadcast Message

**Purpose:** Shareable, forward-friendly message for HR team, existing employees, and referral networks. This is the #1 digital acquisition channel.

**Format:** Image + Text (image provides context when forwarded)

**Message (under 300 characters):**

```
🍳 Kytchens me kaam karo!

💰 ₹14,000 - ₹25,000/month
✅ Full training milegi
✅ PF + ESIC + Incentives
✅ Freshers bhi welcome!

📱 Abhi apply karo 👇
jobs.kytchens.com

Apne dost ko bhi bhejo! 👥
```

**Accompanying Image:**
- Branded card (1080×1080px) with:
  - Kytchens logo
  - "We're Hiring!" badge
  - Salary range (large text)
  - QR code in corner
  - 2-3 benefit icons

**Timing:**
- Best send times: 9-10 AM (before shift) or 7-9 PM (after shift, WhatsApp browsing)
- Sunday mornings get highest engagement (day off)

**Rules:**
- Never send walls of text — gets ignored
- Lead with salary in the first line
- Include "Apne dost ko bhi bhejo" — explicit ask to forward
- Use wa.me link or short URL — not long URLs

### 5.4 Instagram Reel / YouTube Short (15-30 sec)

**Purpose:** Reach workers scrolling Reels/Shorts during breaks and commute.

**Format:** 1080 × 1920px vertical video, 15-30 seconds

**Script Template:**

```
[0-3s]  Hook — Real kitchen footage, fast cuts
        Text overlay: "₹25,000 tak kamaao! 🔥"

[3-8s]  Show kitchen workers in action (real footage)
        Text: "Kytchens me kaam karo"
        Text: "45+ brands, 25+ kitchens"

[8-15s] Benefits — quick bullet points appearing
        "✅ Full Training"
        "✅ PF + ESIC"
        "✅ Freshers Welcome"
        "✅ Career Growth"

[15-20s] Existing worker testimonial (1 line)
         "Maine fresher join kiya, ab Senior TM hoon"

[20-25s] CTA
         "Bio me link se apply karo"
         "Ya QR scan karo" (show QR)

[25-30s] Kytchens logo + tagline
```

**Production Notes:**
- Shoot on phone — polished production feels fake to this audience
- Real workers, real kitchens, real uniforms
- Hinglish voiceover or text overlay (no English narration)
- Upbeat trending audio (check Reels trending sounds)
- Captions always on — many watch without sound

**What NOT to do:**
- Corporate talking-head video
- English narration
- Stock footage
- Longer than 30 seconds

### 5.6 Missed Call Recruitment (Zero-Friction Fallback)

**Purpose:** For candidates who won't scan QR or click links. Eliminates all digital friction.

**How it works:**
1. Candidate sees poster → gives a missed call to the number
2. System auto-sends WhatsApp message with the web app link
3. Candidate clicks link → lands on web app

**Poster text:**
```
Miss call karo: 90821 83834
Hum aapko WhatsApp pe job details bhejenge!
```

**Why this matters:**
- Many blue-collar workers in India are trained on "missed call" interactions (used by banks, telcos, government)
- Zero data cost for the candidate
- Captures phone number even if they don't complete the web form

## 7. QR Code Technical Guide

### How QR → Web App Flow Works

```
Candidate scans QR
        │
        ▼
Phone opens browser → jobs.kytchens.com
        │
        ▼
Landing page loads (< 3 seconds on 4G)
        │
        ▼
Candidate sees salary, taps "Apply Now"
        │
        ▼
Fills screening form (3-4 minutes)
        │
        ▼
Submits → Gets WhatsApp confirmation
        │
        ▼
HR reviews → Calls within 48 hours
```

### QR Code Generation Checklist

- [ ] URL is as short as possible (`jobs.kytchens.com` not `jobs.kytchens.com/recruitment/form/v2`)
- [ ] Error correction level: H (highest — survives 30% damage)
- [ ] Format: SVG for print (scales without pixelation)
- [ ] Minimum print size: 4cm × 4cm
- [ ] White quiet zone: minimum 4 modules wide
- [ ] Add UTM parameters for tracking: `?utm_source=qr&utm_medium=poster&utm_campaign=hiring_mar26`
- [ ] Tested on 3+ low-end Android phones before printing
- [ ] Tested in low-light conditions
- [ ] Fallback URL printed below in plain text
- [ ] Fallback WhatsApp number printed below
- [ ] Scanning instruction ("Yahan scan karo 📱") included

### UTM Tracking for Each Channel

| Channel | UTM Parameters |
|---------|---------------|
| Kitchen poster | `?utm_source=qr&utm_medium=poster&utm_location=[kitchen_id]` |
| Bus stop poster | `?utm_source=qr&utm_medium=outdoor&utm_location=[area]` |
| WhatsApp broadcast | `?utm_source=whatsapp&utm_medium=broadcast` |
| Employee referral | `?utm_source=referral&utm_medium=whatsapp&ref=[employee_name]` |
| Instagram | `?utm_source=instagram&utm_medium=reel` |
| Facebook | `?utm_source=facebook&utm_medium=group` |

## 10. Do's and Don'ts

### Do's ✅
- Always lead with salary — it's the #1 motivator
- Use Hinglish — it's what your audience speaks
- Include a phone number on everything — not everyone will scan
- Use real photos of real kitchens and real workers
- Make everything forward-friendly — assume it will be screenshotted
- Test QR codes on cheap phones before printing
- Track everything with UTM parameters
- Send WhatsApp messages at 9 AM or 7 PM

### Don'ts ❌
- Don't use English-only content — you'll lose 70% of your audience
- Don't use stock photos — workers spot them instantly, trust drops
- Don't make dense, text-heavy posters — keep it to 5-6 lines max
- Don't use small QR codes — minimum 4cm on print
- Don't assume workers know what QR is — always add "Scan karo" instruction
- Don't send long WhatsApp messages — under 300 characters
- Don't skip the fallback phone number — many prefer calling
- Don't use formal Hindi — keep it conversational Hinglish
- Don't post recruitment content only once — repeat weekly
